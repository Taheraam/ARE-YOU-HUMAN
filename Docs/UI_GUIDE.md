# UI GUIDE

## Initial Visual Style
Inspired by:
- Google verification pages
- modern authentication UIs
- clean SaaS dashboards

Visuals:
- white background
- soft shadows
- rounded corners
- blue accents
- minimalist typography

---

## Mid Chaos Visuals

Introduce:
- warning yellow
- terminal aesthetics
- subtle glitches
- shifting layouts
- delayed transitions

---

## Final Chaos Visuals

The UI should appear unstable.

Effects:
- distorted windows
- flickering alerts
- moving buttons
- duplicated elements
- corrupted overlays
- fake system errors
- broken alignment

---

## Animation Style

Beginning:
- smooth
- clean
- polished

Middle:
- resistance
- delay
- tension

End:
- erratic
- aggressive
- unpredictable

---

## Motion Principles
- UI should feel alive
- elements should resist interaction
- motion should increase with chaos
- transitions should become unreliable
