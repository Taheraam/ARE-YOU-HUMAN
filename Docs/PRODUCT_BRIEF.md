# PRODUCT BRIEF

## Project Name
AreYouHuman?

## Core Concept
A reactive CAPTCHA verification website that gradually becomes psychologically unstable.

The site starts as a realistic verification portal but slowly transforms into an absurd, manipulative, self-aware system that questions the user's humanity.

The humor should feel intelligent and unsettling rather than meme-heavy.

## Emotional Progression
1. Familiar
2. Slightly Weird
3. Passive Aggressive
4. Uncomfortable
5. Surreal
6. Total System Collapse

## Design Goals
- Highly interactive
- Frontend only
- Cinematic UX
- Reactive animations
- Strong personality
- Replayability
- Visually escalating chaos

## User Experience Goals
The user should:
- laugh
- feel observed
- become confused
- keep interacting
- want to show others

## Core Features
- Dynamic CAPTCHA challenges
- Chaos escalation system
- Fake AI analysis
- Cursor interaction
- Personality-driven dialogue
- Fake system diagnostics
- Glitch effects
- Dynamic audio
- Ending sequence

## Final Experience
The site eventually stops verifying the user and instead begins verifying itself.
